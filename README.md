# Mini-Projet-A-Language-C

Le projet « Champion des Maths » est un programme codé en C dont l’objectif est de proposer plusieurs exercices de mathématiques, tels que des additions, soustractions, multiplications et divisions. 

À chaque réussite, l’utilisateur qui aura préalablement saisi son nom au lancement du programme remporte des points.

Le programme intègre également un module dédié aux tables de multiplication, permettant soit de s’exercer à les réciter, soit d'afficher une table complète pour révision. La gestion de la progression est assurée par un système de pointeurs et de fichiers : le score de chaque joueur est enregistré dans un fichier .txt externe, permettant ainsi de conserver son historique entre chaque session.
